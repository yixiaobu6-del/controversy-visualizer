# 争议可视化器

> 争议话题结构化分析与可视化工具，正反方观点对比与共识识别

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 争议输入 | 输入争议话题、正方和反方立场定义 |
| 自动生成论点 | 基于内置知识库自动生成正反方论点列表 |
| 共识与分歧 | 标注双方的共识点和分歧点识别 |
| 雷达图对比 | 论点强度/论据数量/逻辑严密性等多维度对比 |
| 导出图片 | 使用 html2canvas 将分析结果导出为 PNG |
| 历史记录 | 保存和回顾历史争议分析结果 |
| 默认示例 | 预置远程办公 vs 办公室办公经典案例 |

## Installation / 安装

无需安装，直接在浏览器中打开 `index.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/controversy-visualizer.git

cd controversy-visualizer
open index.html
```

## Usage / 使用方法

### 基础用法

1. 打开 `index.html`
2. 输入争议话题和双方立场
3. 点击「生成争议分析」
4. 查看正反方论点、共识/分歧点、雷达图对比
5. 点击「导出为图片」保存分析结果

### 默认示例分析

```javascript
// 远程办公 vs 办公室办公
{
  topic: "远程办公 vs 办公室办公",
  proPosition: "远程办公更高效",
  conPosition: "办公室办公更高效",
  proPoints: [
    "节省通勤时间",
    "工作环境自主",
    "全球人才招聘"
  ],
  conPoints: [
    "协作沟通成本高",
    "团队凝聚力弱",
    "工作边界模糊"
  ],
  consensus: ["效率取决于个人", "需要良好管理"],
  divergence: ["协作方式", "文化氛围"]
}
```

### 雷达图维度说明

| 维度 | 说明 | 评分方式 |
|------|------|----------|
| 论点强度 | 论点的说服力和影响力 | 1-10分主观评分 |
| 论据数量 | 支持论点的证据数量 | 自动统计 |
| 逻辑严密性 | 论证逻辑的完整性和严密性 | 1-10分主观评分 |
| 情感共鸣 | 观点引发的情感共鸣程度 | 1-10分主观评分 |
| 实用价值 | 观点的实际应用价值 | 1-10分主观评分 |
