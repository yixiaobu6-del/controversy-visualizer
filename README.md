# 争议可视化器

争论了半天，到底谁站在哪边？
**可视化立场分布，让共识程度一目了然。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 不再"各说各话"

- **立场分布图**：直观展示各立场人数比例
- **共识分析**：关键词重叠度量化，客观判断共识程度
- **投票排名**：热门话题投票排行，发现争议焦点
- **自定义话题**：用户添加话题，按热度排序

## 怎么用

打开 `index.html` → 添加话题 → 查看立场分布
